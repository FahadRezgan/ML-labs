Foodpanda Orders and Customer Dataset
Source: Kaggle – Foodpanda Data Analysis

Dataset Description

This dataset contains 6000 customer records including demographic information, order history, payment methods, loyalty points, ratings, and delivery status.

Machine Learning Problem

This is a Classification problem.

The goal is to predict whether a customer is Active or Inactive (Churned) based on their behavior and order history.

Target Variable

churned

Active

Inactive
